# Сравнительный анализ алгоритмов: Комбинированный метод vs Метод Малышева

Данный репозиторий содержит программное обеспечение на языке MATLAB для воспроизведения численных экспериментов, представленных на Рис. 8 статьи. Программы позволяют сравнить точность и сходимость двух подходов к построению спектральных портретов.

## Описание проекта
В репозитории реализованы алгоритмы для вычисления спектральных портретов двухдиагональных матриц $A$ и $B$. Основное внимание уделяется сравнению:
1. **Комбинированного метода** (алгоритм с разделением пространства).
2. **Метода Малышева**.

## Структура файлов
Репозиторий включает два основных компонента:
* **Скрипт вычислений** — выполняет основной расчет нормы резольвенты и итерационные процессы для матриц $A$ и $B$.
* **`plo.m`** — графический модуль, отвечающий за визуализацию результатов (построение графиков спектральных портретов, ошибок и количества итераций, аналогично Рис. 8).
